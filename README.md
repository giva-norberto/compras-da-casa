# Compras da Casa

Aplicativo pessoal para controle de estoque doméstico e lista inteligente de compras.

## Funcionalidades

- Controle de estoque atual
- Estoque mínimo
- Cálculo automático da quantidade a comprar
- Lista automática de compras
- Confirmação da quantidade realmente comprada
- Atualização automática do estoque
- Aba IA com alertas simples
- Interface feita para celular
- Dados salvos no Firebase Firestore

## Estrutura do projeto

```text
compras-da-casa/
│
├── index.html
└── README.md
```

Projeto feito em HTML, CSS e JavaScript puro, sem framework.
